Machine Learning

Machine Learning is allowing for the speed of solutions to problems within many different fields. From Physics and Mathematics to Biology and Industry, fields are getting revolutionised by the introduction of machine learning.

Industry

In Industry, new metals are usually laboriously made by scientists with very common metals like iron and combinining them with more expensive metals to see the effects. But after training a machine learning model on hundreds of different metal alloys and their properties the model could then predict and suggest different combinations of metals to allow for production of useful properties. In this case invariability of size when temperatures are changed greatly. Metals can contract and expand due to heat and in industry. If a metal were to expand too much the system it is a part of could break down. This improvement speeds up the process of finding new alloys to be used in industry as the labour of scientists is not as necessary for testing. Usually analysis takes years, with the new machine learning tool OmniFold it takes minutes. And although processing multiple datapoints had been possible previously, it had not been able to account for distortions in the detector now that it can. Mathematics

Mathematics has had its matrix multiplication speed increased with reinforcement learning. The efficiency of matrix multiplication has been increased shaving off 2 more multiplications from matrix multiplication reducing computation time. AlphaTensor was the network behind this discovery, although 2 steps may not seem like much, when millions of matrix multiplications are taking place 2 multiplications add up much faster. The matrix multiplication was treated like a single player game and the goal given to the computer was to find the least possible steps to perform the full multiplication which it has succeeded at.
